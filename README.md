Custom Markdown
====

文書を記述するための軽量マークアップ言語であるMarkdownの表示を好みで変更できるエディタ

## Feature
* 入力を即座にMarkdown形式で変換して出力
* 以下のスタイルを自分好みに編集可能
  - 見出し1(H1)
  - 見出し2(H2)
  - コード
  - 強調
* 出力方法としてWebPageとPDF(未完成)が可能
* 前回入力した内容を記録(session)

## Demo
http://custom-markdown.mybluemix.net/

## Usage

```
$ npm start
```

## Install

